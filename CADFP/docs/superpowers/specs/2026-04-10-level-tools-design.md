# Level Tools Design

**Date:** 2026-04-10

**Goal**

在现有“测量工具”模块中补齐 `圆形水平仪` 与 `条形水平仪` 两个页面，视觉尽量贴近 Figma 节点 `109:1501` 和 `111:1566`，并基于 iPhone 真机重力传感器提供可实时工作的水平测量能力。

**Constraints**

- 保持现有 SwiftUI 导航结构，不破坏已完成的直尺与量角器页面。
- 不引入第三方传感器或滤波依赖，直接使用 Apple `CoreMotion`。
- 目录页继续复用 `MeasurementCatalogScreen`，进入水平仪页后隐藏导航栏和 tabBar。
- 圆形水平仪尽量贴 Figma：白底、蓝色圆盘、十字分隔线、单个气泡点。
- 条形水平仪当前 Figma 节点只给了极简读数态，因此允许在不破坏整体留白风格的前提下补一条真实可用的水平条。

**Recommended Approach**

采用“共享重力引擎 + 两种几何投影”的方案：

- 新增 `LevelMotionEngine`，统一管理 `CMMotionManager`、更新频率、低通滤波和设备朝向下的主测量轴选择。
- 新增 `LevelGeometry`，把 `gravity.x / y` 转成：
  - 圆形水平仪气泡偏移
  - 条形水平仪气泡偏移
  - 显示角度
- 圆形页严格对齐 Figma 的静态构图，只让气泡点随重力移动。
- 条形页保留 Figma 的居中大号角度数字，同时补一条低干扰水平条与气泡，避免页面只有数字没有仪表反馈。

**Architecture**

1. 路由层

- `ContentView` 新增：
  - `measurementCircularLevel`
  - `measurementBarLevel`

2. 目录层

- `MeasurementCatalogScreen`
  - 将圆形水平仪与条形水平仪从占位按钮改成真实 `NavigationLink`

3. 传感器层

- `LevelMotionEngine`
  - 负责启动 / 停止 `deviceMotion`
  - 维护最新重力向量
  - 根据设备当前界面方向自动切换条形水平仪的主测量轴
  - 对连续更新做轻量滤波，减少气泡抖动

4. 几何层

- `LevelGeometry`
  - 负责圆形气泡的安全半径裁切
  - 负责条形气泡在水平条上的位置映射
  - 负责角度格式化与“接近水平”判定

5. 页面层

- `CircularLevelScreen`
  - 白底
  - 大号蓝色圆盘
  - 十字分割线
  - 实时气泡
  - 右上关闭按钮
- `BarLevelScreen`
  - 白底
  - 中央角度读数
  - 极简水平条
  - 水平条中的气泡或游标
  - 右上关闭按钮

**Motion Mapping**

- 圆形水平仪：
  - 使用 `gravity.x` 与 `gravity.y`
  - 把二维重力偏差投影到圆盘内，气泡越接近圆心越水平
- 条形水平仪：
  - 自动根据当前设备方向切换主轴
  - 竖屏优先测左右倾斜
  - 横屏优先测上下倾斜
  - 主轴角度使用 `atan2` / 倾角换算，显示为 `-9°` 这类整数或 1 位小数格式

**Filtering Strategy**

- 采用低通滤波平滑重力更新，避免气泡高频抖动。
- 条形和圆形共用同一份滤波后的重力值，保证读数和气泡反馈一致。
- 接近 0° 时允许更稳的吸附体验，但不做强制锁死，避免“突然跳正”的假精度感。

**Error Handling**

- 模拟器或无传感器环境下显示友好提示，不伪装成真实水平仪。
- 页面首次出现时自动开始更新；离开页面时停止更新，避免无意义电量消耗。
- 若 `deviceMotion` 不可用，则展示静态空态文案。

**Success Criteria**

- 测量工具目录页点击 `圆形水平仪` 和 `条形水平仪` 都能进入真实页面。
- 圆形页在真机上可看到气泡随设备倾斜实时移动。
- 条形页在竖屏 / 横屏之间会自动切换主测量轴，读数方向正确。
- 两页视觉与 Figma 的白底、关闭按钮、核心构图保持一致。
- 重力更新稳定，无明显闪烁或方向反转错误。
- 几何映射和角度格式化有单元测试覆盖。

**Risks**

- `CoreMotion` 在不同机型上的噪声略有差异，滤波参数需要保守，不宜过重。
- 条形水平仪的 Figma 节点信息较少，补充的水平条需要保持克制，避免偏离整体视觉。
- 设备界面方向与用户握持方向并不总是完全一致，需要用当前界面方向做合理兜底。

**Out of Scope**

- 震动校准提示
- 零点手动校准
- 历史记录、截图、分享
- AR 或相机辅助水平测量
